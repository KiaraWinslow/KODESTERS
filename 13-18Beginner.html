Programs

The Coder School

The Coder School is an after-school program for kids and parents living in the silicon valley or adjacent area. Check out their website to see if there is a program near you!

Black Girls Code

Black Girls Code is an organization aiming to increase the number of women of color in the tech industry. They provide free programs for women of color and african american youth. Check out their website to see if you are eligable!

If you know how to read you can learn how to code. Their online coding tutorials with easy-to-follow instructions, immediate feedback, and a tested curriculum take anyone from non-technical to “I can code.”
